Application to simply track personal and incubating projects. 
Good place to stuff in all the great ideas you want to catch up 
later while keeping focus on your active projects.

This application features:
- Sortable project list
- Log (diary)
- Self organising roadmap with minor and major milestones
- Feature and issue issues tracking

Watch a presentation on YouTube: http://www.youtube.com/watch?v=xA_KckRueLQ

You can try the 30 day trial package or buy the packaged full edition in the
Ubuntu software store.
